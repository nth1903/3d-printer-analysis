# 3D Printer Analysis

## Dataset

This project uses the 3D Printer Dataset from Kaggle:
https://www.kaggle.com/datasets/afumetto/3dprinter

## Description

This project analyzes the impact of 3D printing parameters on product quality:

* Surface roughness
* Tensile strength
* Elongation

## Tools

* R programming language
* Libraries: tidyverse, caret, randomForest, ggcorrplot

## Files

* data.csv: dataset
* analysis.R: R code
* report.pdf: final report

## Methods

* Descriptive statistics
* Data visualization
* Random Forest

## Objective

To understand relationships between printing parameters and output quality.
